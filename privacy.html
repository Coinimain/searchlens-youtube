PRIVACY POLICY

SearchLens for YouTube

Last updated: July 2026

Overview

SearchLens for YouTube is a browser extension that adds advanced search controls to YouTube search-result pages. This Privacy Policy explains what information the extension accesses, how that information is used, and what is stored when you use SearchLens.

Information SearchLens processes

To provide its search filtering and sorting features, SearchLens reads information that is already displayed on YouTube search-result pages. Depending on the feature being used, this may include:

  • Video titles
  • Channel names
  • Video duration information
  • Visible view counts
  • Video links and YouTube video identifiers
  • Visible search-result metadata

This information is used to apply the filters, sorting options, and search controls selected by the user.

User activity and clicked videos

SearchLens can detect when you click a YouTube video link while using YouTube search results. This is used for the optional feature that can hide watched or previously clicked videos.

When this feature is used, SearchLens may store the associated YouTube video identifier locally in Chrome extension storage so the extension can recognize that video later.

SearchLens does not record the contents of videos you watch, your viewing duration, comments, messages, account credentials, or other activity outside the functionality described in this policy.

Web history

For the optional watched or previously clicked video filtering feature, SearchLens may locally remember identifiers for YouTube videos that you clicked from search-result pages. the optional watched or previously clicked video filtering

This information is used only to help SearchLens identify and hide previously clicked videos when that option is enabled.

Information stored by the extension

SearchLens uses Chrome's extension storage to remember settings and preferences so they can persist between browser sessions. Stored information may include:

  • Selected SearchLens filter settings
  • Date-range preferences
  • Panel and launcher state
  • Floating launcher position and visibility
  • First-run interface state
  • Locally remembered YouTube video identifiers for the optional previously clicked video feature

Local processing and data transmission

SearchLens processes the YouTube search-result information needed for its features within the user's browser.

SearchLens does not send extension settings, SearchLens filter data, locally remembered video identifiers, or the YouTube page information it processes to a developer-operated server.

SearchLens does not require a SearchLens account and does not sell user data.

How information is used

Information accessed or stored by SearchLens is used only to provide the extension's search-enhancement features, including:

  • Filtering YouTube search results
  • Sorting currently loaded search results
  • Applying keyword, title, channel, duration, and view-count filters
  • Hiding selected result types
  • Remembering user preferences
  • Supporting the optional previously clicked video filter

SearchLens does not use this information for advertising, creditworthiness, lending decisions, or purposes unrelated to the extension's search functionality.

Permissions and YouTube access

SearchLens requires access to YouTube pages so it can add the SearchLens interface, read visible search-result information needed for user-selected filters, detect navigation within YouTube, and apply filtering and sorting to search results.

SearchLens also uses Chrome's storage permission to save extension settings and locally remembered information needed for its features.

Data sharing and transfer

SearchLens does not sell user data or transfer the information it handles to third parties for advertising, marketing, data brokerage, creditworthiness, or unrelated purposes.

Third-party services and external links

SearchLens may provide optional links to third-party websites, such as Ko-fi or the Chrome Web Store. These links open only when selected by the user.

Third-party websites operate under their own terms and privacy policies. SearchLens does not control their data practices.

Changes to this policy

This Privacy Policy may be updated if SearchLens features, permissions, or data-handling practices change. The latest version will be published on this page.

Contact and support

For questions about SearchLens or this Privacy Policy, use the support method provided on the Chrome Web Store listing or visit the SearchLens support page.
