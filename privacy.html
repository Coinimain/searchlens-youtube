PRIVACY POLICY

SearchLens for YouTube

Last updated: July 2026

Overview

SearchLens for YouTube is a browser extension that adds advanced search controls to YouTube search-result pages. This policy explains how the extension handles information when you use it.

Information stored by the extension

SearchLens uses Chrome's local extension storage to remember settings such as filter preferences, panel state, launcher position, and other extension options.

If you use the watched or previously clicked video filtering feature, SearchLens may also store YouTube video identifiers locally so it can recognize those videos later.

Data transmission

SearchLens does not send extension settings or locally stored video identifiers to a developer-operated server. The extension does not require a SearchLens account.

Permissions and page access

SearchLens runs on YouTube pages so it can add its search interface and filter the results displayed on the page. It also uses Chrome storage permission to save extension preferences.

Third-party services

SearchLens may provide optional links to third-party websites, such as Ko-fi or the Chrome Web Store. Those websites operate under their own terms and privacy policies. SearchLens does not control their data practices.

Changes to this policy

This policy may be updated when SearchLens features or data-handling practices change. The latest version will be published on this page.

Contact and support

For questions about SearchLens, use the support method provided on the Chrome Web Store listing or visit the SearchLens support page.
